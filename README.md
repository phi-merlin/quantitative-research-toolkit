# Quantitative Research Toolkit

**Statistical learning, time series modeling, and network analysis applied to complex datasets**

---

## Overview

This repository gathers a set of projects focused on **extracting structure, signals, and insights from high-dimensional data**.

The work covers:

* time series modeling and forecasting
* unsupervised learning and feature extraction
* network analysis and structural dynamics

These methods are broadly applicable to complex systems, including **financial markets, macroeconomic data, and trading analytics**.

---

## Projects

### 📈 Time Series Modeling

* Trend and seasonality decomposition
* Polynomial and autoregressive (AR) modeling
* Residual analysis and model evaluation

**Key focus:** separating deterministic structure from stochastic dynamics to improve forecasting.

*Relevance:* applicable to macroeconomic modeling, price dynamics, and systematic signal construction.

---

### 🔍 Unsupervised Learning & Feature Extraction

* Dimensionality reduction using PCA
* Clustering with K-means
* Nonlinear embeddings (T-SNE, MDS) for structure visualization
* Feature selection via regularized logistic regression

**Key focus:** identifying latent structure in high-dimensional datasets and building reduced representations.

*Relevance:* useful for regime detection, cross-sectional analysis, and feature engineering.

---

### 🌐 Network Analysis & Structural Dynamics

* Graph construction and analysis
* Centrality measures (degree, betweenness, eigenvector)
* Identification of key nodes and influence
* Analysis of dynamic network evolution
* Hub/authority structure in directed networks

**Key focus:** understanding how influence, information, and connectivity evolve in complex systems.

*Relevance:* applicable to market structure analysis, counterparty networks, and contagion dynamics.

---

## Methods & Tools

* Python (NumPy, scikit-learn, matplotlib)
* Statistical modeling and machine learning
* Graph and network analysis

---

## Objective

The objective of this work is to develop a **robust quantitative toolkit** for analyzing complex datasets, with a focus on:

* signal extraction
* structural understanding
* predictive modeling

These approaches are directly transferable to **quantitative research and trading analytics**.

---

## Notes

* Projects are presented with a focus on **methodology and interpretation**, rather than domain-specific context
* Emphasis is placed on **modeling rigor, scalability, and interpretability**
